# 壁面 計測デモアプリ

## 1. 概要

写真1枚から壁の実寸（幅・高さ・面積）を算出し、遮音材の必要枚数を自動計算するデモアプリ。
本アプリは平面前提の簡易測量ツールとする。

---

## 2. 想定ユースケース

- 簡易見積もりデモ
- 遮音材販売時の必要数量シミュレーション
- 写真ベース測量SaaSのプロトタイプ

---

## 3. 前提条件

- 壁は平面であること
- 画像内に既知サイズ物体（例：A4用紙 210mm × 297mm）を含めること
- 斜め撮影可（透視変換で補正）

---

## 4. 機能要件

### 4.1 画像アップロード

- JPG / PNG 対応
- ブラウザ内処理（サーバー送信なし）

### 4.2 スケール設定

- A4用紙の4点をユーザーがクリック
- ピクセル距離からスケール係数算出

スケール計算式：

    scale(mm_per_pixel) = 実寸(mm) / ピクセル距離

### 4.3 壁領域指定

- 壁の四隅をクリック
- 台形状態でも可

### 4.4 透視変換

- OpenCV.js の getPerspectiveTransform()
- warpPerspective()

### 4.5 寸法算出

    幅(mm) = ピクセル幅 × scale
    高さ(mm) = ピクセル高さ × scale
    面積(㎡) = 幅 × 高さ / 1,000,000

### 4.6 遮音材枚数計算

例：910mm × 1820mm の場合

    材料面積 = 0.91 × 1.82
    必要枚数 = ceil(壁面積 / 材料面積 × 1.05)

※5%余裕を自動加算
